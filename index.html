Human Scream Detection and Analysis for Controlling Crime Rate

I developed an intelligent acoustic surveillance system that detects human screams from real-time audio to automatically flag potential emergencies. The system leverages Mel-Frequency Cepstral Coefficients (MFCC) for feature extraction and a novel hybrid SVM-MLP model for high-accuracy classification, aiming to reduce crime by enabling faster response times.

The Application of Logistic Regression and Discriminant Analysis in Classifying Corporate Bankruptcy Status.

This project's objective was to build a robust early-warning system for predicting corporate bankruptcy. Using the R programming environment, I implemented and compared two powerful statistical classification methods. I applied Logistic Regression to assess the probability of failure and Discriminant Analysis to establish clear decision boundaries between healthy and at-risk companies. This dual-model approach resulted in a reliable predictive tool that not only forecasts bankruptcy but also identifies the key financial indicators signaling distress.
